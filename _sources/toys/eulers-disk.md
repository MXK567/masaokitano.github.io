# オイラーの円盤

```{figure} eulers-disk.mp4
:class: controls
```

オイラーの円盤 (Euler's disk) とよばれる,
風変わりなコマを作ってみよう.

材料は3.5インチハードディスクの
プラッター(円盤)である.
ある程度, 質量があった方がよいので,
3～4枚を接着剤で貼り合わせる.
(ずれないように位置合わせを慎重に.)

ハードディスクの分解ができない場合は市販品もある.

- [シミュレーション](https://demonstrations.wolfram.com/DiskRollingOnAHorizontalPlane)

- [理論](https://arxiv.org/pdf/physics/0008227)
